# SiLog — AI-Powered SIWES Logbook Assistant

SiLog helps Nigerian university students write perfect SIWES industrial attachment logbooks with AI assistance.

## ✨ Features

- Daily activity logging with AI expansion
- Activity bank for overflow entries
- 24-week dashboard with tracking
- Mock defense preparation
- University template library
- Export to PDF/Word

## 🚀 Get Started

Visit [SiLog-app.vercerl.app](https://silog-app.vercel.app) to sign up and start using the app.

## Tech Stack

Built with Next.js, React, Tailwind CSS, Supabase, and Claude AI.
